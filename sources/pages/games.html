Games and Instructions

Snake Game

Classic snake game where you guide a line (snake) to a dot (food) and the goal is to make the snake longer for every dot it eats. You use the arrow keys to control the snake..

Tic Tac Toe

A simple game where u can play against the Computer or a person. The AI of the computer you play again is based on minimax algorithm..

Pong

Classic Pong game. Use the arrow up and down to move the slider..

Memory

A simple Dinosaur memory game. Has Dinosaur audio built in..

The Oregon Trail

Classic text based game from the 1990. Game is loaded directly from archive.org.
