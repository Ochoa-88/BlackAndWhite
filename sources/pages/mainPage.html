Welcome to Black & White

Introduction

This website is part of a final grade for "Web Technologies". Web Techologies is a module provide by Edinburgh Napier University.

Black & White is a site that is designed mostly with HTML and CSS with added functionality using JavaScript. The site is to display and launch games to show the capabilites of Javascript within a webpage. The site is dedicated for a quick introduction to web design.

At the top of the page there are links to the Main Page (where you currently are) and other links to description of the games and references.

To the left side you will find the buttons to launch the games available on this site. Starting off with a classic Snake game. And a more advanced game, the classic Oregon Trail.
